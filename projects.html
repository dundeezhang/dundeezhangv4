Bank Console Application

This project was my first somewhat useful project made using C++. I made this to mostly just learn and reinforce my understanding on C++ filesystems, class, and the basics in general. This program has password protected logins for users and staff. Depending on the password, the program will prompt the user what to do depending on their "type of user." This program can create new users, delete users, wipe whole system, change selective information, save user password for ease of use, and change user passwords.
